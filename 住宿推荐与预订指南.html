Claude in Chrome 预订 Prompts

以下 prompt 可直接复制到 Claude in Chrome 中使用。先登录 Booking.com 或 Airbnb，然后让 Claude 帮你操作筛选和比价

Prompt 1：Booking.com 批量搜索（推荐先用这个）

请帮我在 Booking.com 上依次搜索以下6个城市的酒店，每个城市找出评分8.0+、三人可住、价格最优的3个选项： 1. 巴黎 Le Marais区：3.31入住 4.03退房，3人，预算€120-180/晚 优先看：Hotel Jeanne d'Arc, Hotel Bastille Speria, Hotel Turenne Le Marais 2. 科隆 大教堂旁：4.03入住 4.04退房，3人，预算€80-130/晚 优先看：CityClass Hotel am Dom, SMARTY Cologne Dom Hotel 3. 慕尼黑 火车站附近：4.04入住 4.07退房，3人，预算€100-180/晚 优先看：Aloft Munchen, Boutique Hotel Splendid 4. 格林德瓦：4.07入住 4.09退房，3人，预算CHF150-250/晚 优先看：Eiger Lodge, Eiger Mountain & Soul Resort 5. 威尼斯 Cannaregio区：4.09入住 4.11退房，3人，预算€100-170/晚 优先看：Carnival Palace, Eurostars Residenza Cannaregio 6. 罗马 Monti区：4.11入住 4.13退房，3人，预算€90-150/晚 优先看：Condominio Monti, Hotel Colosseum 每个酒店请告诉我：实际价格、评分、是否有三人间、是否有电梯、是否含早餐、取消政策。最后给我一个总价对比表。

Prompt 2：Airbnb 搜索（如果偏好公寓）

请帮我在 Airbnb 上搜索以下住宿，每个城市找出评分4.8+、评论50+的最佳3个选项： 要求统一标准： - 整套公寓/房屋（非合住） - 至少2间卧室或1卧+沙发床（3人住） - 必须有电梯或在低楼层（2楼以内） - 有WiFi和洗衣机 城市和日期： 1. 巴黎 3区/4区(Le Marais)：3.31-4.03，€120-180/晚 2. 慕尼黑 市中心5km内：4.04-4.07，€100-160/晚 3. 威尼斯 Cannaregio/San Marco：4.09-4.11，€100-160/晚 4. 罗马 Monti/Termini附近：4.11-4.13，€90-140/晚 （科隆1晚和格林德瓦2晚建议用酒店，不用搜Airbnb） 每个选项请截图并告诉我：价格、评分、到主要景点步行距离、房东回复率。

Prompt 3：Google Maps 验证位置

我准备预订以下酒店，请帮我在 Google Maps 上验证每个酒店的位置： [粘贴你选中的酒店名称和地址] 对每个酒店，请告诉我： 1. 到最近地铁站的步行距离和时间 2. 到当天要去的第一个景点的路线和时间 3. 周边500米内有没有超市和餐厅 4. 街景看起来是否安全整洁
