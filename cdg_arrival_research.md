# Day 1 - 巴黎CDG机场到达攻略 (2026年3月31日)

> 航班到达时间：18:15 | 3人家庭（含年长父母）| 携带行李

## 1. CDG机场航站楼信息

### 亚洲国际航班通常使用哪个航站楼？

- **Terminal 2E/2F**：大多数亚洲长途航班到达此处，包括：
  - Air France（亚洲航线）→ Terminal 2E
  - 中国国航 (Air China) → Terminal 2E
  - 中国东方航空 (China Eastern) → Terminal 2E
  - 国泰航空 (Cathay Pacific) → Terminal 1
  - 新加坡航空 (Singapore Airlines) → Terminal 1
  - 大韩航空 (Korean Air) → Terminal 2E
- **Terminal 1**：星空联盟成员航空公司（如国泰、新航、汉莎等）
- **务必在出发前确认具体航站楼**，以航司/机票确认为准

### 入境+取行李通常需要多久？

| 环节 | 预估时间 |
|------|---------|
| 下飞机到入境柜台 | 10-15分钟 |
| 入境排队+过关 | 20-45分钟（非欧盟护照通道） |
| 等行李 | 15-30分钟（CDG行李出名慢） |
| 海关通道 | 5分钟 |
| **总计** | **50-95分钟** |

**预估离开机场时间：19:15 - 19:45**

> ⚠️ 提示：CDG机场的行李转盘效率在欧洲大型机场中偏慢，30分钟等行李很常见。18:15到达的航班避开了早晨的高峰到达时段，排队会相对好一些。

## 2. CDG到巴黎市中心交通方式对比

目的地：Notre-Dame / Le Marais 区域（右岸 Rive Droite）

### 方案A：官方出租车（推荐 ⭐）

| 项目 | 详情 |
|------|------|
| 价格 | **€56 固定费率**（到右岸 Rive Droite，含Le Marais） |
| 时长 | 45-60分钟（取决于交通） |
| 说明 | 不加行李费、不加人头费（4人以内）、含机场接车费 |
| 在哪坐 | 到达大厅出口，跟 "Taxi" 标志走到官方出租车排队区 |

> ⚠️ **务必走官方出租车站**（有穿制服的工作人员），不要理睬大厅里主动上来拉客的人，那些是非法出租车。

### 方案B：Welcome Pickups 私人接机

| 项目 | 详情 |
|------|------|
| 轿车 Sedan | **€63**（可坐4人） |
| 商务车 Minivan | **€108**（空间更大，适合多行李） |
| 服务特色 | 司机在航站楼内举牌接机，含90分钟免费等待，固定价格无隐藏费 |
| 预订 | [welcomepickups.com/paris/airport-transfer](https://www.welcomepickups.com/paris/airport-transfer/) |

### 方案C：KiwiTaxi 私人接机

| 项目 | 详情 |
|------|------|
| 起步价 | **从$77起（约€70+）** |
| 服务特色 | 航站楼内举牌接机，含90分钟免费等待，固定价格 |
| 预订 | [kiwitaxi.com](https://kiwitaxi.com/en/france/de+gaulle+airport-%3Eparis) |

### 方案D：GetTransfer 私人接机

| 项目 | 详情 |
|------|------|
| 价格 | 需在官网询价，通常€60-90（轿车） |
| 特色 | 竞价模式，多个司机报价，可选最优 |
| 预订 | [gettransfer.com](https://gettransfer.com/en/directions/france) |

### 方案E：RER B 火车

| 项目 | 详情 |
|------|------|
| 价格 | **€14/人**（3人共€42）+ Navigo Easy卡 €2/人 |
| 时长 | 约37分钟到 Châtelet-Les Halles |
| 末班车 | 工作日末班约22:50从CDG出发 |
| 购票 | 纸质票已取消，需购买Navigo Easy卡（€2）在自助机上充值，或用手机App（Bonjour RATP） |

> ⚠️ **不推荐RER B给带行李的年长父母**：需要在站台上下楼梯，车厢内拥挤（即使是晚间），安全风险较高（RER B沿线治安一般），拖行李不方便。

### 方案F：Roissybus ❌ 已停运

| 项目 | 详情 |
|------|------|
| 状态 | **2026年3月1日起永久停运**，运营34年后关闭 |
| 替代 | Line 9517 替代，但经停 Saint-Denis Pleyel，不直达市中心 |

### 🏆 最终推荐

**对于携带行李的年长父母，晚间到达的情况，强烈推荐：**

1. **首选：官方出租车（€56）** — 最简单、价格合理、固定费率、无需预订、随到随走
2. **次选：Welcome Pickups（€63轿车/€108商务车）** — 如果希望有人在航站楼内举牌接机，减少找出租车的麻烦，提前预订更安心

**不推荐 RER B** — 虽然便宜（3人€42+€6卡费），但需要扛行李上下楼梯，晚间安全性不如私人接送，对老人不友好。

## 3. 晚餐推荐（Le Marais / Notre-Dame 附近）

预计到达酒店/住宿时间：**约20:15-20:45**，需要21:00后仍开放的餐厅。

### 推荐餐厅

#### ① Les Philosophes
- **地址**：28 Rue Vieille du Temple, 75004 Paris（Le Marais核心地段）
- **营业时间**：9:00 - 2:00（凌晨2点关门，完全不用担心太晚）
- **人均**：€25-35
- **推荐菜品**：
  - 法式洋葱汤 (Soupe à l'oignon gratinée)
  - 鸭胸 (Magret de canard)
  - 每日套餐 Menu du Jour（前菜+主菜+甜点，约€25-30，性价比最高）
- **特色**：传统巴黎小酒馆风格，露天座位可以看街景，氛围极好
- **语言**：有英文菜单

#### ② Au Bougnat
- **地址**：26 Rue Chanoinesse, 75004 Paris（Notre-Dame旁边的小街上）
- **营业时间**：午餐+晚餐至约22:30
- **人均**：€25-40
- **推荐菜品**：
  - 红酒炖牛肉 (Bœuf bourguignon) — 招牌菜
  - Aveyron香肠 — 餐厅经典
  - 焦糖布丁 (Crème brûlée)
- **特色**：离Notre-Dame步行5分钟，温暖的传统法式小馆，非常地道

#### ③ Bouillon République（备选，稍远但超值）
- **地址**：39 Boulevard du Temple, 75003 Paris
- **营业时间**：全天不间断供餐（持续服务制，不怕到太晚）
- **人均**：**€15-20**（巴黎最高性价比法餐之一）
- **推荐菜品**：
  - 焗蜗牛 (Escargots de Bourgogne)
  - 油封鸭腿 (Confit de canard)
  - 法式甜点拼盘
- **特色**：Bouillon是巴黎传统平价法式食堂概念的复兴，菜品正宗、分量足、价格极低。排队可能较长，但翻桌快

## 4. 第一晚实用贴士

### 时差管理（Jet Lag）
- 中国 → 巴黎时差：**-6小时**（夏令时，3月31日已切换夏令时）
- 到达时巴黎18:15 = 北京时间次日0:15，身体已经很疲惫
- **建议**：
  - 到达当晚不要太早睡，尽量撑到巴黎时间22:00-23:00再休息
  - 吃一顿正式晚餐有助于调整生物钟
  - 第二天早上设闹钟，不要睡到自然醒（否则时差调不过来）
  - 避免咖啡因（到达当晚）

### ⚠️ 安全提示
- **机场大厅**：不要理睬任何主动搭讪的人，尤其是"需要出租车吗"之类的
- **出租车**：只走官方出租车站排队
- **Le Marais区域**：巴黎最安全的街区之一，晚间散步也比较安心
- **随身物品**：护照和钱包放在贴身位置，不要放在外套口袋
- **地铁/RER**：如果要坐地铁，注意扒手，特别是在Châtelet-Les Halles和Gare du Nord
- **手机**：到达后买一张欧洲SIM卡或开通国际漫游，确保Google Maps可用

### 夏令时提醒
- **2026年3月29日（周日）**凌晨2:00，欧洲切换夏令时（钟表拨快1小时）
- 你们3月31日到达时已经是夏令时，天黑时间约为20:00-20:15
- 到达时天还亮着，有助于缓解初到的不安感
